# Stock-Data
Python Stock Data Webapp -- using Dash

Python Version
 Python 3.8.1 (v3.8.1:1b293b6006, Dec 18 2019, 14:08:53)

Dependencies:
 pip3 install yfinance
 pip3 install Dash
 pip3 install pandas-datareader
 pip3 install dash-bootstrap-components
 pip3 install lxml
